# Filter

Filter help users more easily use them to select lists. Filter will transmit different data when user switch them.

## Properties

<PropsTable of={Filter} />

## Filter usage

<Playground>
    <Filter list={['01','02']} defaultList={['01', '02']} />
</Playground>

## Support multiple selection

<Playground>
    <Filter list={['01','02']} defaultList={['01', '02']} multiple/>
</Playground>

<Playground>
    <Filter list={{1:'01',2:'02',3:'03'}} defaultList={['01', '02','03']} multiple object/>
