## API

### 静态方法：
+ start: 开启进度条
+ done(status): 进度条完成
+ inc: 随机增加进度, 但是不会增加到 100%
+ set: 进度条进度, (0 - 1)
+ configure(options): 进度条配置 参数见下方
+ remove: 移除进度条元素
