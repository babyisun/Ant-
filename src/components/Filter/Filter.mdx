# Filter

Filter组件帮助你更好的进行条件筛选，摒弃select需要下拉选择的繁琐，让筛选项拥有即视感，支持单选和多选两种模式。

## 代码演示

简单示例
<Playground>

        { label: '分类1', value: '1' },

    />
</Playground>

在Ant-Design表单中使用

<Playground>
    {
    () => {
      class Container extends Component {
        onSubmit(e){
            e.preventDefault();
            const { form: { validateFields } } = this.props;
            validateFields((err, values) => {
                if (!err) {
                    console.log('onSubmit:', values);
                }
            });
        };

        render() {
          const { form } = this.props;
          return (
            <Form onSubmit={e => this.onSubmit(e)}>
              <Form.Item label="表单示例">
                {
                form.getFieldDecorator('filterValues', {
                  initialValue: ['2'],
                })(<Filter options={[
                    { label: '分类1', value: '1' }, 
                    { label: '分类2', value: '2' },
                    { label: '分类3', value: '3' }]}
                    multiple
                    onChange={(value, options)=>console.log('onChange：', value)}
                     />)
                }
              </Form.Item>
              <Button htmlType="submit" icon="search" label="&nbsp;">
                    搜索
              </Button>
            </Form>
          );
        }
      }

      const FormDemo = Form.create()(Container);
      return <FormDemo/>
    }
  }
</Playground>
