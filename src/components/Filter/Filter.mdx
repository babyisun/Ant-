# Filter

Filter组件帮助你更好的进行条件筛选，摒弃select需要下拉选择的繁琐，让筛选项拥有即视感，支持单选和多选两种模式。

## 代码演示

简单示例
<Playground>
    <Filter options={[
        { label: '分类1', value: '1' }, 
        { label: '分类2', value: '2' },
        { label: '分类3', value: '3' }]} 
    />
</Playground>

带有默认值的多项筛选
<Playground>
    <Filter options={[
        { label: '分类1', value: '1' }, 
        { label: '分类2', value: '2' },
        { label: '分类3', value: '3' }]}
        defaultValue={['1', '2']}
        multiple
     />
</Playground>
