# Filter

Filter组件帮助你更好的进行条件筛选，摒弃select需要下拉选择的繁琐，让筛选项拥有即视感，支持单选和多选两种模式。

## 代码演示

简单示例
<Playground>
    <Filter options={[
        { lable: '分类1', value: '1' }, 
        { lable: '分类2', value: '2' },
        { lable: '分类3', value: '3' }
    ]} />
</Playground>

数据格式为数组的多选模式
<Playground>
    <Filter options={['01','02']} defaultValue={['01', '02']} multiple/>
</Playground>

数据格式为对象的多选模式

<Playground>
    <Filter options={{1:'01',2:'02',3:'03'}} defaultValue={['01', '02','03']} multiple/>
</Playground>
